# Aseprite — Professional Pixel Animation and Sprite Creation for Indie Developers

## About Aseprite

Aseprite provides a complete professional pixel art and sprite animation environment for indie developers, pixel artists, and game designers. Aseprite combines pixel-perfect drawing tools, timeline animation, and color palette management. Aseprite supports onion skinning, frame tagging, and looping previews. Aseprite integrates export formats for game engines, sprite sheets, and GIF animations. Aseprite enables creators to produce hand-crafted pixel art with retro aesthetic and modern workflows. Aseprite is trusted by indie game teams, pixel artists, jam participants, and retro game developers worldwide.
